# pullrequest

> A GitHub App built with [Probot](https://github.com/probot/probot) that a &#x27;pullrequest&#x27;

## Setup

```sh
# Install dependencies
npm install

# Run the bot
npm start
```

## Docker

```sh
# 1. Build container
docker build -t pullrequest .

# 2. Start container
docker run -e APP_ID=<app-id> -e PRIVATE_KEY=<pem-value> pullrequest
```
